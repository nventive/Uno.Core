## Release 1.27.0

### Features

 * Direct compatibility with projects targeting uap 10.0.17763

### Breaking changes

 * Removed CategoryAttribute and ObfuscationAttribute from the lib. If your project uses them, change the target SDK to UAP 10.0.17763 or higher.
